## Directory Purposes

**`src/`:**

- Purpose: Source modules, each self-contained with dedicated TypeScript config
- Contains: 9 modules (types, 4 transformations, 1 execution engine, 2 convenience wrappers, 1 barrel export)
- Key files: `src/index.ts` (barrel export), each module's `index.ts`

**`src/types/`:**

- Purpose: Core type system and sentinels
- Contains: `Reducer<A, B>`, `Transducer<A, B>`, `Reduced<T>`, `reduced()`, `isReduced()`
- Key files: `src/types/index.ts`, `src/types/tsconfig.json`
- Module refs: Referenced by all other modules

**`src/map/`, `src/filter/`, `src/take/`, `src/drop/`:**

- Purpose: Individual transducer implementations
- Contains: Single function exporting a transducer factory
- Pattern: Each module has `index.ts` (implementation) and `tsconfig.json` (references `types`)
- Dependencies: All depend on `src/types/`

**`src/transduce/`:**

- Purpose: Core execution layer ‚Äî applies composed transducers to iterables
- Contains: `transduce(xform, reducer, init, iterable)` function
- Key logic: Iteration loop, early termination via `isReduced()` check
- Dependencies: `src/types/`

**`src/into/`, `src/sequence/`:**

- Purpose: Ergonomic APIs for common operations
- Contains: Wrapper functions around `transduce`
- Dependencies: `src/types/`, `src/transduce/`
- `into()`: Pre-configures array push reducer
- `sequence()`: Wraps `into()` with empty array initialization

**`dist/`:**

- Purpose: Build artifacts (generated by tsup)
- Contains: ESM bundles (`*.js`), type declarations (`*.d.ts`), sourcemaps (`*.js.map`, `*.d.ts.map`)
- Generated: Yes ‚Äî created by `yarn build`
- Committed: No ‚Äî in `.gitignore`

- `oxlintrc.json` ‚Äî Linter rules (correctness=error, suspicious/pedantic=warn)
- `.editorconfig` ‚Äî Formatting: 2-space indent, LF, UTF-8

**Core Logic:**

- `src/types/index.ts` ‚Äî Reducer and Transducer type definitions
- `src/map/index.ts`, `src/filter/index.ts`, `src/take/index.ts`, `src/drop/index.ts` ‚Äî Transformation implementations

**Testing:**

- Not detected in codebase (test files not found)

## Naming Conventions

**Files:**

- `index.ts` ‚Äî Sole implementation file in each module (exports single function or types)
- `tsconfig.json` ‚Äî One per module, extends `../../tsconfig.base.json`
- Lowercase module directories: `map`, `filter`, `take`, `drop`, `types`, `transduce`, `into`, `sequence`

**Directories:**

- Lowercase, single word per directory
- One function/export per directory (except `types` which exports multiple)
- Pattern: `src/[function-name]/index.ts`

**Functions:**

- camelCase: `map`, `filter`, `take`, `drop`, `transduce`, `into`, `sequence`, `reduced`, `isReduced`
- Descriptive, single responsibility per function

**Types:**

- PascalCase: `Reducer`, `Transducer`, `Reduced`
- Generic parameters: Single letter (`<A>`, `<B>`, `<R>`) for transformations, semantically clear in context

**Imports:**

- Use `.ts` extensions in all imports (e.g., `from "../types/index.ts"`)
- Reflects `allowImportingTsExtensions` and `verbatimModuleSyntax` compiler options

## Where to Add New Code

**New Transducer Transformation:**

- Create new module: `src/[name]/`
- Create `src/[name]/index.ts` ‚Äî implement function returning `Transducer`
- Create `src/[name]/tsconfig.json` ‚Äî reference `types`
- Add to `tsconfig.json` project references
- Add entry point to `tsup.config.ts`
- Export from `src/index.ts`
- Example pattern (follow `map.ts`):

  ```typescript
  import type { Reducer, Transducer } from "../types/index.ts";

  export function myTransform<A, B>(config: any): Transducer<A, B> {
    return <R>(rf: Reducer<R, B>): Reducer<R, A> =>
      (acc: R, input: A): R =>
        rf(acc, transformedInput);
  }
  ```

**New High-Level API:**

- Similar structure to `into` or `sequence`
- Depends on: `src/types/`, `src/transduce/`
- Location: `src/[api-name]/index.ts`
- Must wrap or enhance `transduce` function

**Utilities/Helpers:**

- Not currently present; would be added to `src/types/index.ts` if needed
- Or create `src/utils/` following same module pattern

**Type Extensions:**

- Modify `src/types/index.ts` only
- Re-exported from `src/index.ts`
- Update TypeScript configs if adding new generic constraints

## Import Paths

**From package consumers:**

- Main: `import { map, filter, transduce } from "transducer-ts"`
